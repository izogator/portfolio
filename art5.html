04.01.01 - SPRITE ANIMATION

So far I enjoyed working in Photoshop and Illustrator, but I loved this task. I got to draw a lot and create a fun character in Illustrator. My character is a mix of different ideas - the first sketch came up while a conversation about the movie "Birdman". Colours and textures I picked after looking at different birds. After creating the first illustration, some friends said that he looks a bit creepy, so I decided to use pastel colours for the background to make it look a bit more friendly. The final sprite animation was created with HTML and CSS keyframes.
